# 🚀 新闻重写系统 - 快速开始指南

## 🎯 系统简介

这是一个专业的科技新闻自动获取和重写系统，能够：
- 📰 自动获取最新科技新闻（支持NewsAPI、HackerNews、RSS源）
- ✍️ 将新闻重写为吸引人的自媒体内容
- 🖼️ 为每篇文章生成详细的图片建议
- 🏷️ 自动提取关键词和分类标签
- 📝 输出标准markdown格式文章

## 🚀 立即体验

### 1. 运行演示（无需API密钥）
```bash
python demo.py
```

这将使用预设的演示数据生成3篇重写后的文章，让您快速了解系统功能。

### 2. 查看生成的文章
演示完成后，查看 `output/` 目录中的markdown文件：
- 每篇文章都包含吸引人的标题
- 完整的自媒体风格内容结构
- 详细的图片建议
- 相关标签和来源信息

## ⚙️ 配置真实新闻源

### 1. 获取API密钥

#### NewsAPI（推荐，免费）
1. 访问 [NewsAPI.org](https://newsapi.org/)
2. 注册账号获取免费API密钥
3. 免费版每天100次请求，足够个人使用

#### OpenAI API（可选）
1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 创建API密钥用于高级内容重写

### 2. 配置环境变量
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，添加你的API密钥
NEWSAPI_KEY=your_newsapi_key_here
OPENAI_API_KEY=your_openai_key_here  # 可选
```

### 3. 运行完整系统
```bash
cd src
python main.py
```

## 📋 使用示例

### 基本使用
```bash
# 获取5篇最新科技新闻并重写
python main.py

# 获取3篇AI相关新闻
python main.py -n 3 -t AI

# 获取10篇科技新闻
python main.py --max-articles 10

# 指定主题过滤
python main.py -t "人工智能" -n 5
```

### 命令行参数
- `-n, --max-articles`: 最大文章数量 (默认: 5)
- `-t, --topic`: 主题过滤关键词
- `-c, --config`: 配置文件路径

## 🔧 自定义配置

### 修改新闻源
编辑 `config/config.yaml`：
```yaml
rss_feeds:
  - name: "Your Tech Blog"
    url: "https://yourblog.com/feed/"
```

### 调整重写风格
```yaml
rewriter:
  style: "自媒体"
  tone: "专业但通俗"
  target_reading_time: "3-5分钟"
```

### 过滤关键词
```yaml
filters:
  keywords_include:
    - "AI"
    - "人工智能"
    - "区块链"
  keywords_exclude:
    - "广告"
    - "推广"
```

## 🐛 常见问题

### Q: API密钥错误怎么办？
A: 检查 `.env` 文件中的密钥是否正确，确认API密钥有效且未过期。

### Q: 获取不到新闻怎么办？
A: 检查网络连接，某些API可能需要代理。也可以先运行演示模式测试系统。

### Q: 生成的内容质量不满意？
A: 可以调整 `config/config.yaml` 中的过滤条件和重写参数。

### Q: 想要更多新闻源？
A: 在配置文件中添加更多RSS源，或者扩展代码支持其他API。
