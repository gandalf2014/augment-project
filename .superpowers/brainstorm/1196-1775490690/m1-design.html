📊 数据模型变更

-- 新增 notebooks 表
CREATE TABLE notebooks (
  id INTEGER PRIMARY KEY,
  name TEXT NOT NULL,
  icon TEXT DEFAULT '📁',
  sort_order INTEGER DEFAULT 0,
  created_at TEXT DEFAULT datetime('now'),
  updated_at TEXT DEFAULT datetime('now')
);

-- memos 表新增字段
ALTER TABLE memos ADD COLUMN notebook_id INTEGER;
ALTER TABLE memos ADD COLUMN is_archived INTEGER DEFAULT 0;

📁 笔记本功能

  • 单层结构 — 不支持嵌套，简单直观
  • 侧边栏显示 — 笔记本在上，标签在下，分离显示
  • 排序 — 按最近更新时间，活跃笔记本在前
  • 默认笔记本 — 系统内置「未分类」，新备忘录默认进入
  • CRUD — 创建、重命名、删除（删除时备忘录移至未分类）
  • 计数显示 — 每个笔记本显示备忘录数量

📦 归档功能

  • 特殊笔记本 — 「已归档」显示在侧边栏底部
  • 归档操作 — 右键菜单/操作按钮选择「归档」
  • 恢复操作 — 在已归档列表中选择「恢复到...」
  • 视觉区分 — 已归档笔记本半透明显示
  • 不参与搜索 — 全局搜索默认不搜归档内容

🔄 API 变更

GET /api/notebooks 获取笔记本列表
POST /api/notebooks 创建笔记本
PUT /api/notebooks/:id 重命名笔记本
DELETE /api/notebooks/:id 删除笔记本（备忘录移至未分类）
GET /api/memos?notebook=... 按笔记本筛选（新增参数）
POST /api/memos/:id/archive 归档备忘录
POST /api/memos/:id/restore 恢复备忘录
